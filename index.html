Glowing Mouse Pointer

Move your mouse around to see the glowing effect

How it works

The glowing effect follows your mouse pointer using JavaScript. The glow is created with CSS radial gradients.

Customization

You can easily customize the glow color, size, and intensity by modifying the CSS variables.

Browser Support

This effect works in all modern browsers that support CSS gradients and JavaScript.

Performance

The effect is optimized for smooth performance with minimal impact on browser resources.
